## 1. 다재약물 임상 정의 및 분류 기준

### 1.1 권장 정의

| 구분 | 기준 | 비고 |
|------|------|------|
| **다재약물 (Polypharmacy)** | 동일 기간(90일) 내 **5종 이상** 만성질환 치료 약물 동시 복용 | ATC 5단계 고유 성분 기준 |
| **과다재약물 (Excessive Polypharmacy)** | **10종 이상** 동시 복용 | 고위험군 판정의 보조 기준 |

### 1.2 약물 카운트 산정 기준

**포함 대상**
- 만성질환 치료 목적의 지속 처방 약물 (투약일수 8일 이상)
- 전신 작용 약물

**제외 대상**
- PRN(필요시 복용) 약물
- 단기 처방 (투약일수 ≤ 7일): 급성기 감기약, 단기 항생제 등
- 외용제 (ATC D 피부과 계열 전신작용 없는 것)
- 점안제, 점이제 (ATC S 감각기관)
- 백신 (ATC J07)
- 진단용 약제 (ATC V)

### 1.3 동시복용 판정 알고리즘

```
약물A 복용기간: [처방시작일, 처방시작일 + 총투약일수]
약물B 복용기간: [처방시작일, 처방시작일 + 총투약일수]

중첩일수 = max(0, min(A종료일, B종료일) - max(A시작일, B시작일))

중첩일수 ≥ 7일  → 동시복용으로 판정
중첩일수 1~6일  → 약물 전환(switching)으로 판정, 동시복용 제외

예시:
  약물A: 2026-01-01 처방, 30일 → 복용기간 01/01~01/30
  약물B: 2026-01-15 처방, 60일 → 복용기간 01/15~03/15
  중첩기간: 01/15~01/30 = 16일 → 동시복용 ✓
```

**관찰 윈도우**
- **90일 윈도우** (Primary): 만성 다재약물 판별
- **30일 윈도우** (Secondary): 급성기 약물 중첩 탐지 (항생제 + 만성약 DDI)

## 2. 약물상호작용(DDI) 분류 체계

### 2.1 DDI 심각도 4단계 분류

| 등급 | 정의 | 임상 결과 | 개입 수준 |
|------|------|-----------|-----------|
| **Contraindicated** | 절대 병용금기 | 생명 위협적 부작용 발생 확실 | 즉시 개입 필수 (처방 중단) |
| **Major** | 생명위협 또는 영구 손상 가능 | 입원, 생명위협, 영구 손상 가능 | 적극적 개입 (처방 재검토) |
| **Moderate** | 기존 질환 악화 또는 추가 치료 필요 | 임상적으로 의미 있는 부작용 | 모니터링 + 환자 교육 |
| **Minor** | 경미한 영향, 임상적으로 대체로 무의미 | 경미한 증상 또는 영향 없음 | 정보 제공 |

### 2.2 권장 DDI 참조 데이터베이스

| 순위 | DB | 장점 | 활용 방안 |
|------|-----|------|-----------|
| **1순위** | **HIRA DUR DB** | 국내 보험 약품코드 직접 매핑, 규제기관 공인 | Primary 참조 |
| **2순위** | **DrugBank** | 구조화 우수, ATC 코드 기반, 학술 신뢰도 | 보완 참조 |
| **3순위** | **식약처 의약품안전나라** | 국내 허가사항 기반 병용금기 정보 | 추가 검증 |
| **참고** | **FDA FAERS** | 부작용 시그널 탐지 | 신호 감지용 |
| **참고** | **Lexicomp / Clinical Pharmacology** | 임상 현장 표준 DB | 불명확 사례 검토 |

**이중 참조 체계 권장**: HIRA DUR을 1차로 사용하고 DrugBank로 보완. 두 DB의 불일치 시 심각도 높은 쪽 채택(Conservative approach).

### 2.3 DDI 매트릭스 테이블 스키마

```sql
CREATE TABLE ddi_matrix (
    drug_a_atc   VARCHAR(7),    -- ATC 5단계 코드
    drug_b_atc   VARCHAR(7),    -- ATC 5단계 코드
    severity     ENUM('Contraindicated','Major','Moderate','Minor'),
    mechanism    VARCHAR(200),   -- 예: 'CYP3A4 inhibition'
    clinical_effect VARCHAR(300),-- 예: '출혈 위험 증가'
    evidence_level VARCHAR(20),  -- 'Established','Probable','Suspected'
    source       VARCHAR(50),    -- 'HIRA_DUR','DrugBank','Both'
    last_updated DATE,
    PRIMARY KEY (drug_a_atc, drug_b_atc)
);
-- 주의: (A,B)와 (B,A) 모두 저장하거나 조회 시 양방향 조인 처리
```

## 3. 중복약물(Duplicate Therapy) 탐지 기준

### 3.1 3단계 탐지 전략

| Level | 기준 | 예시 | 우선순위 |
|-------|------|------|---------|
| **Level 1** | ATC 5단계 동일 (완전 동일 성분) | 두 기관에서 amlodipine 처방 | 최우선 탐지 |
| **Level 2** | ATC 4단계 동일 (동일 약리 소분류) | amlodipine + nifedipine (둘 다 C08CA) | 2순위 |
| **Level 3** | ATC 3단계 동일 (동일 치료목적) | ACE억제제 + ARB (C09A + C09C) | 예외 규칙 적용 후 판정 |

### 3.2 중복약물 허용 예외 규칙 (E1~E9)

**기본 예외 (E1~E5)**

| 코드 | 규칙명 | 해당 ATC | 임상 근거 |
|------|--------|----------|-----------|
| E1 | 항고혈압제 다제병용 | C02-C09 | JNC/ESC 가이드라인 권장 병용 (기전이 다른 2~3종 병용이 표준) |
| E2 | 당뇨병 다제병용 | A10 | 기전이 다른 병용 허용: 메트포르민+SGLT2i+DPP4i 등 |
| E3 | 흡입제 병용 (천식/COPD) | R03 | ICS+LABA+LAMA 3제 병용이 GINA/GOLD 가이드라인 표준 |
| E4 | 진통제 단계적 병용 | N02, M01 | WHO 진통제 사다리(Step Ladder) 접근법 |
| E5 | DAPT (이중항혈소판) | B01AC | 관상동맥 시술(PCI) 후 표준 치료: aspirin + P2Y12 억제제 |

**정신과 약물 예외 (E7~E9)**

| 코드 | 규칙명 | 해당 ATC | 임상 근거 |
|------|--------|----------|-----------|
| E7 | 항우울제 보조 병용 | N06A | 치료 저항성 우울증에서 두 계열 병용 (예: SSRI + SNRI 제외, SSRI + Mirtazapine은 허용) |
| E8 | 항정신병약 병용 (전문의 처방) | N05A | 치료저항성 조현병에서 전문의 판단 하 2종 병용 허용 |
| E9 | 기분안정제 + 항정신병약 | N05A + N03A | 양극성 장애 급성기/유지기 치료 표준 조합 |

> **주의**: E7~E9는 전문의 처방 확인이 전제. 일반의/비정신과 처방 시에는 예외 미적용.

### 3.3 약물 전환(Switching) vs 중복 구분

```
같은 ATC 3단계 내 약물 교체:
  - A 종료일 근처에서 B 시작 + 이후 A 재처방 없음 → 전환 (중복 아님)
  - A와 B가 30일 이상 동시 지속 → 중복약물

중첩일수 역치 적용:
  - 1~6일 중첩: 전환으로 판정
  - 7일 이상 중첩: 동시복용으로 판정
```

## 4. 환자 위험도 분류 기준

### 4.1 위험도 4단계 분류 기준 (상세)

#### 🔴 고위험군 (Red) - 즉각 개입

다음 조건 중 **하나 이상** 해당 시:

- Contraindicated DDI **1건 이상** 보유
- Major DDI **3건 이상** 보유
- **Triple Whammy** 해당 (ACEi/ARB + K-sparing이뇨제 + NSAIDs)
- 10종 이상 약물 복용 + Major DDI **1건 이상** 포함
- **고위험 약물**(항응고제, 항혈소판제, 인슐린, 항경련제, 면역억제제, 항암제) 포함 + Major DDI
- 75세 이상 + 5종 이상 + 신기능/간기능 저하 추정 약물 포함
- QT 연장 약물 3종 이상 동시 복용

#### 🟡 중위험군 (Yellow) - 정기 모니터링

다음 조건 중 **하나 이상** 해당 시 (고위험 기준 미충족):

- Major DDI **1~2건** 보유
- Moderate DDI **2건 이상** 보유
- 동일 성분 중복처방 (Level 1 중복) 확인
- **3개 이상** 의료기관에서 동시 처방 (doctor shopping 의심)
- 10종 이상 약물 복용 (DDI 없더라도)
- 고위험 약물 포함 + Moderate DDI

#### 🟢 저위험군 (Green) - 정보 제공/교육

다음 조건 해당 시 (고·중위험 기준 미충족):

- Minor DDI만 보유
- 5종 이상이나 DDI 없음
- ATC 3단계 수준 중복만 존재 (임상적 병용 허용 범위)
- 동일 ATC 4단계 중복이나 예외 규칙(E1~E9) 해당

#### ⚪ 정상 (Normal) - 개입 불필요

- 다재약물 기준 미충족 (5종 미만) + DDI 없음

### 4.2 개입 수준 제안

| 위험군 | 개입 방법 | 빈도 | 담당 |
|--------|-----------|------|------|
| 고위험 (Red) | 처방 의사 직접 연락 + 환자 약물상담 | 즉시 (탐지 시) | 전문약사/의사 |
| 중위험 (Yellow) | 처방 검토 알림 + 환자 교육 자료 발송 | 월 1회 | 건보 상담원 |
| 저위험 (Green) | 자동 안내 문자/앱 알림 | 분기 1회 | 자동화 |

## 5. 고위험 약물 조합 상세 (Top 10 DDI)

반드시 100% 탐지해야 하는 필수 DDI 목록:

### DDI #1: 와파린/DOAC + NSAIDs
- **위험**: 출혈 위험 급증 (위장관출혈, 뇌출혈)
- **기전**: 항응고 효과 증강 + 위점막 손상
- **ATC**: B01AA03 (와파린), B01AF (DOAC) + M01A (NSAIDs)
- **심각도**: Major (와파린), Contraindicated (일부 DOAC 조합)
- **ADR 프록시**: K92(위장관출혈), I60-I62(뇌출혈), D68(응고장애)

### DDI #2: Clopidogrel + Omeprazole/Esomeprazole
- **위험**: 항혈소판 효과 최대 50% 감소
- **기전**: CYP2C19 억제 → 클로피도그렐 활성화 차단
- **ATC**: B01AC04 (clopidogrel) + A02BC01 (omeprazole), A02BC05 (esomeprazole)
- **심각도**: Major
- **대안**: Pantoprazole (CYP2C19 영향 적음) 또는 H2 차단제

### DDI #3: ACEi/ARB + K-sparing 이뇨제 + NSAIDs (Triple Whammy)
- **위험**: 급성신부전 (Acute Kidney Injury)
- **기전**: 신장 혈류량 감소 (3중 타격)
- **ATC**: C09A/C09C + C03DA + M01A
- **심각도**: Contraindicated (3가지 동시 병용)
- **ADR 프록시**: N17(급성신부전), E87(전해질이상)
- **특이사항**: 셋 중 둘만 병용해도 위험하나, 셋이 동시일 때 급격히 위험 증가

### DDI #4: 디곡신 + 아미오다론/베라파밀
- **위험**: 디곡신 혈중농도 상승 → 디곡신 독성 (부정맥, 오심)
- **기전**: P-gp 억제 + 신장 배설 감소 (아미오다론), CYP3A4 억제 (베라파밀)
- **ATC**: C01AA05 (digoxin) + C01BD01 (amiodarone) / C08DA01 (verapamil)
- **심각도**: Major
- **ADR 프록시**: I49(부정맥), R11(구역/구토)
- **모니터링**: 디곡신 혈중농도 모니터링 필수 (치료범위: 0.5~2.0 ng/mL)

### DDI #5: 메토트렉세이트 + 트리메토프림/설파메톡사졸
- **위험**: 골수억제, 범혈구감소증
- **기전**: 엽산 길항 기전 상승 작용
- **ATC**: L01BA01/L04AX03 (MTX) + J01EA01/J01EE01 (TMP/SMX)
- **심각도**: Contraindicated
- **ADR 프록시**: D61(재생불량성 빈혈), D70(무과립구증)

### DDI #6: SSRI + MAO 억제제
- **위험**: 세로토닌 증후군 (생명 위협)
- **기전**: 세로토닌 과잉 → 신경독성
- **ATC**: N06AB (SSRI) + N06AF/N06AG (MAOi)
- **심각도**: Contraindicated
- **ADR 프록시**: G25(이상운동), R56(경련)
- **주의**: 약물 중단 후 washout period 필요 (fluoxetine: 5주)

### DDI #7: SSRI + Triptan
- **위험**: 세로토닌 증후군 (가능성)
- **기전**: 세로토닌 수용체 활성화 중첩
- **ATC**: N06AB + N02CC
- **심각도**: Major
- **FDA**: 2006년 경고문 발령 → 2010년 완화 (임상적 필요 시 신중 병용 가능)

### DDI #8: 리튬 + NSAIDs/이뇨제
- **위험**: 리튬 독성 (치료범위 좁음: 0.6~1.2 mEq/L)
- **기전**: NSAIDs → 신장 PGE2 억제 → 리튬 재흡수 증가
- **ATC**: N05AN01 (lithium) + M01A (NSAIDs) / C03 (이뇨제)
- **심각도**: Major
- **ADR 프록시**: G92(독성뇌병증), R11, R55

### DDI #9: QT 연장 약물 다중 병용
- **위험**: 치명적 심장 부정맥 (Torsades de Pointes, 심실세동)
- **기전**: QT 간격 연장 → 심실 재분극 지연
- **주요 QT 연장 약물**:
  - 항정신병약: haloperidol, quetiapine, risperidone
  - 항부정맥약: amiodarone, sotalol
  - 마크로라이드: azithromycin, clarithromycin
  - 퀴놀론: ciprofloxacin, levofloxacin
  - 항히스타민: (처방전 발급 일부)
- **기준**: QT 연장 약물 3종 이상 동시 복용 시 고위험
- **참조**: CredibleMeds QTDrugs List (crediblemeds.org)

### DDI #10: Statin + 마크로라이드 항생제 (Clarithromycin/Erythromycin)
- **위험**: 횡문근융해 (Rhabdomyolysis)
- **기전**: CYP3A4 억제 → statin 혈중농도 급상승
- **해당 statin**: simvastatin, atorvastatin (lovastatin)
- **예외**: pravastatin, rosuvastatin은 CYP3A4 비의존적 → 위험 낮음
- **ATC**: C10AA + J01FA

## 6. CYP450 효소 기반 상호작용 체계

### 6.1 주요 CYP 효소 역할

| 효소 | 주요 기질 약물 | 임상 중요도 |
|------|--------------|------------|
| **CYP3A4** | 스타틴, 칼슘채널차단제, 면역억제제, 일부 항생제, 항진균제 | 가장 많은 약물 대사 (처방 약물의 약 50%) |
| **CYP2D6** | 항우울제(SSRI, TCA), 항정신병약, 일부 베타차단제, 오피오이드 | 개인 유전 다형성 중요 |
| **CYP2C9** | 와파린, NSAIDs, 일부 경구혈당강하제 (sulfonylurea), 페니토인 | 항응고 모니터링 핵심 |
| **CYP2C19** | PPI(오메프라졸 등), 클로피도그렐, 일부 항우울제 | 동양인 poor metabolizer 비율 높음 |
| **CYP1A2** | 테오필린, 카페인, 클로자핀, 일부 항정신병약 | 흡연이 유도제로 작용 |

### 6.2 CYP 강도별 피처 분류 (25개)

**CYP3A4 피처 (5개)**
- `cyp3a4_substrate_count`: 기질 약물 수
- `cyp3a4_strong_inhibitor_count`: 강력 억제제 수 (ketoconazole, clarithromycin 등)
- `cyp3a4_moderate_inhibitor_count`: 중등 억제제 수 (diltiazem, verapamil 등)
- `cyp3a4_inducer_count`: 유도제 수 (rifampin, carbamazepine 등)
- `cyp3a4_interaction_risk`: 기질 + 강력억제제 동시 존재 쌍 수

**CYP2D6, CYP2C9, CYP2C19, CYP1A2 각각 동일 구조** (각 5개 × 4 = 20개)

## 9. 정신과 약물 예외 규칙 (E7~E9)

정신과 약물은 임상적 병용이 허용되는 경우가 많아 별도 예외 규칙을 적용:

### E7: 항우울제 보조 병용 허용 조건

**허용**: 치료저항성 우울증에서 기전이 다른 병용
- SSRI + Mirtazapine (일부 evidence 있음)
- SSRI + Bupropion (기전 다름)
- SSRI + TCA 소량 (수면 목적, 전문의 처방)

**불허**: 동일 계열 병용
- SSRI + SNRI (세로토닌 과잉 위험)
- TCA + TCA
- MAOi + 모든 세로토닌 작용제

**판정 조건**: 정신건강의학과 전문의 처방 확인 (요양기관 진료과목 코드 확인)

### E8: 항정신병약 병용 (N05A) 예외 조건

**허용**: 치료저항성 조현병, 전문의 처방 확인 시
- 장기지속형 주사제(LAI) + 경구 항정신병약 (전환 기간)
- 1차/2차 세대 조합 (clozapine + aripiprazole 등, 근거 있음)

**불허**: 유사 기전 동일 계열 2종 병용 (근거 없는 병용)

### E9: 기분안정제 + 항정신병약 예외 조건

**허용**: 양극성 장애 표준 치료
- 리튬 + 항정신병약 (급성 조증 또는 유지 치료)
- 발프로에이트 + 항정신병약
- 라모트리진 + 항정신병약 (우울 삽화 예방)

**주의**: 리튬 + 항정신병약 병용 시 QT 연장 모니터링 필요

## 11. 건보 청구 데이터 한계 및 대응

### 11.1 주요 한계점

| 한계 | 설명 | 영향 |
|------|------|------|

### 11.2 대응 방안

| 한계 | 대응 방안 |
|------|-----------|
| OTC 미포함 | 항응고제/리튬 처방 환자에 "OTC NSAIDs 병용 주의" 경고 플래그 자동 부착 |
| 복약순응도 | 처방일수 기반 MPR(Medication Possession Ratio) 보조 피처로 활용 |
| 처방일수 중복 | overlap 알고리즘 + 중첩 7일 이상 역치로 단순 재처방 제외 |
| 용량 | DDD(Defined Daily Dose) 기준 표준화 |
| 입원/외래 | T20 진료형태 코드로 구분, 입원 처방은 퇴원 후 외래 연속성 확인 |
| 모델 결과 고지 | 결과 보고 시 OTC/한약 정보 미포함 사실 명시 |

## 12. 골든 데이터셋 임상 판정 기준서 (v0.1)

### 12.1 리뷰어 자격 요건

- 약사 면허 소지자 (경력 5년 이상) 또는 임상약학 전문약사
- DUR 업무 경험 2년 이상 권장
- 본 기준서 교육 이수 후 calibration 테스트 통과 (테스트 세트 기준 Kappa ≥ 0.75)

### 12.2 판정 절차

```
1단계: 환자 처방 목록 확인 (ATC 코드, 동시복용 기간)
2단계: DDI 체크 (DDI 매트릭스 조회)
        → 최고 심각도 DDI 식별
3단계: 중복약물 체크 (Level 1 → 2 → 3 순서)
        → 예외 규칙(E1~E9) 적용 여부 판단
4단계: 고위험 약물 포함 여부 확인
5단계: 위험도 등급 판정 (Red/Yellow/Green/Normal)
6단계: 판정 근거 기록 (주요 DDI, 중복약물, 특이사항)
7단계: 불확실 케이스는 보류 표시 → 3차 전문가 합의
```

### 12.3 판정 기준 문서 목록 (구축 예정)

### 12.4 어노테이션 도구 요구사항 (데이터엔지니어에게 전달)

리뷰어가 사용할 임상 검토 뷰(Clinical Review View)에 포함되어야 할 정보:

## 14. 오버라이드 규칙 (O1)

### O1: 임상 전문가 판단 우선 규칙

모델 예측 등급보다 임상 전문가(약사/의사) 판단이 우선하는 경우:

**적용 조건**:
- 임상 전문가가 모델 예측과 다른 등급으로 오버라이드한 경우
- 오버라이드 사유 기록 필수 (임상적 판단 근거)

**오버라이드 사유 분류**:
- OVR-01: 환자 개별 임상 상황 (신기능, 간기능 실측치 반영)
- OVR-02: 이미 모니터링 중인 DDI (혈중 농도 정기 측정 중)
- OVR-03: 데이터 오류 (처방 종료 약물이 데이터에 남아있음)
- OVR-04: 임상 가이드라인 예외 사항
- OVR-05: 기타 (자유 텍스트)
